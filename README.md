## 🛠️ Tecnologias Utilizadas

| Camada      | Tecnologia              | Descrição                          |
|-------------|-------------------------|------------------------------------|

## 🚀 Como Executar o Projeto

### 🔧 Pré-requisitos
- Node.js v18 ou superior  
- Git instalado  

### ⚙️ Passo a passo
```bash
# 1. Clone o repositório
git clone https://github.com/GSOUZA11/produtos-app.git 

# 2. Acesse a pasta
cd produtos-app

# 3. Instale as dependências
npm install

# 4. Inicie o servidor
node server.js
